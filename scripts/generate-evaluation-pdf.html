Purpose: Function review, process map, campaign-kit vision, multi-reference strategy, market positioning.

Executive summary. The technical pipeline (Nano Banana + Seedance 2.0 via fal.ai) is strong (~8/10). The product experience still sells separate steps (image, then video) instead of one campaign outcome (~5/10 beginner success). Video reference matching and some image paths under-deliver when inputs, modes, or prompts misalign. The highest-leverage direction is universal Campaign Kit: concept + product (+ optional references) → a set of marketing assets, with structured re-edit—not “100% perfect AI in one click.”

2. Current user process (as built)

SETUP (template, copy, style) ↓ IMAGE STEP ├─ promo-ai → 1 product photo → 1 AI image ├─ reference-concept → product + 1 reference IMAGE → 1 AI image └─ paper-layout → compositor (exact text) ↓ VIDEO STEP (if combined / video-only) ├─ image-to-video → 1 keyframe → ~8s motion (reference MP4 IGNORED) ├─ reference-concept → 1 product + 1 reference MP4 → Seedance R2V └─ multi-angle → product + 2–3 angle photos ↓ DONE → download (+ BGM on video)

Why it feels “image by image, video by video”

  • Three separate creative choices; no single campaign brief drives all outputs.
  • Combined flow defaults to image → video from that still; reference MP4 easy to upload but skipped if mode is wrong.
  • No one-click kit: no 5–6 carousel, no caption pack, no “story + reel + still” bundle.
  • User mental model: I gave concept + product + reference — give me a campaign.
    App mental model: Run image API, then video API.

5. One upload vs more images & references

Seedance 2.0 (fal) limits

InputLimitPrompt tags
ImagesUp to 9@Image1 … @Image9 — product identity, angles
VideosUp to 3, combined 2–15 seconds@Video1 … — motion, editing, hands
AudioUp to 3, combined ≤15s@Audio1 — rhythm (optional)

When more helps

  • Same product, 2–4 angles (front, on wrist, macro) → better identity in motion.
  • Style board as @Image2 while @Image1 = product.
  • Multi-angle mode in app (API already accepts multiple images).

When more hurts

  • Contradictory refs (doodle ref + marble still + different products).
  • Screen recordings with IG UI.
  • Ref video > 15s without trimming in CapCut.

10. Post-generation editing

AI will not be 100% right first time. Layers that fit this project:

Positioning: AI drafts the campaign → user polishes in CapCut/Canva. Market: 「AI 起稿，你執字執片」— not 「一鍵完美」.

Multi-AI for carousel (5–6 posts)

  • LLM — slide storyboard + 繁體 per slide
  • Nano Banana Pro — product + reference ad per slide
  • Compositor — exact text slides
  • Ideogram (optional) — text-heavy promo
  • Seedance 2.0 — one Reel from best slide or ref video
